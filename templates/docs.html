Connecting to Haggle is simple and easy. Just sign up with our api to obtain a client ID and secret. Using this, you can connect using them as parameters. Here is an example:(this is where I'll put an example)

The client intending to access protected Haggle resources registers with Haggle. Haggle issues a Client ID and Client Secret.

The client uses the client ID and Client Secret to send an authorization request. Haggle authenticates the client, grants the authorization request and returns a authorization code. The client sends the client ID and secret along with the authorization code to request for an access token from Haggle. Haggle authenticates the client, validates the authorization code and returns an access token.
